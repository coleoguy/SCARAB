# TOB Phase 2 — Tier-3 Sanger Mining Runbook

Mine ~10,000–30,000 Coleoptera species from GenBank using SuperCRUNCH, align per locus,
build constrained per-family trees. All jobs run on Grace in `$SCRATCH/tob/`.

---

## Prerequisites

- Phase 1 backbone tree complete and approved (stored as `$SCRATCH/tob/backbone/backbone.tre`)
- SuperCRUNCH installed via `01_install_supercrunch.sh`
- Bouchard↔NCBI reconciliation table at `$SCRATCH/tob/data/bouchard2024_ncbi_reconciliation.csv`
- Standard loci reference sequences at `$SCRATCH/tob/data/locus_references.fasta`
  (one reference per locus: COI, 16S, 18S, 28S, CAD, EF1a, ArgK, RNApol2, wingless)

## Directory layout on Grace

```
$SCRATCH/tob/

├── envs/

├── SuperCRUNCH/                        # git clone
├── raw_seqs/

├── filtered_seqs/
│   └── {FAMILY}/                       # per-family post-BLAST extracts
├── alignments/
│   └── {FAMILY}/                       # per-locus alignments + trimmed
├── supermatrices/
│   └── {FAMILY}/                       # per-family supermatrix + partition file
├── trees/
│   └── {FAMILY}/                       # per-family IQ-TREE output
├── backbone/
│   └── backbone.tre                    # Phase 1 constraint tree
├── qc/
│   └── {FAMILY}_qc.tsv                 # per-family quality reports
├── logs/                               # SLURM logs
└── no_data_families.txt                # families with no NCBI taxid
```

## Step-by-step

| Step | Script | Where | Notes |
|------|--------|-------|-------|
| 1 | `01_install_supercrunch.sh` | login node | Once only; internet required |
| 2 | `02_pull_per_family.sh` | login node | Pulls FASTA from NCBI Entrez; ~hours |
| 3 | `03_filter_loci.sh` | login node or interactive | BLAST-based locus extraction |
| 4 | `sbatch 04_align_per_family.slurm` | Grace SLURM | Array: one task per family |
| 5 | `python 05_concat_per_family.py` | login node | After array completes |
| 6 | `sbatch 06_iqtree_per_family.slurm` | Grace SLURM | Array: one task per family |
| 7 | `python 07_quality_gate.py` | login node | Review before advancing |

## Parameters requiring Heath confirmation

- `MINCOLS`: minimum alignment columns after trimAl (currently 100)
- `OCCUPANCY`: occupancy threshold (currently 0.30 per workflow_v1.md)
- `MIN_TAXA_FAMILY`: minimum taxa per family to attempt a tree (currently 4)
- Backbone tree path: `$SCRATCH/tob/backbone/backbone.tre` (update after Phase 1)
- `LOCUS_REFERENCES`: path to the 9 reference sequences used for BLAST extraction

## Taxonomy notes

- 443 families: exact NCBI match
- 335 families: missing from NCBI (fossil/ichnotaxon) → written to `no_data_families.txt`
- 12 families: conflict (Bouchard separates; NCBI lumps as subfamilies):
  Cicindelidae, Colonidae, Disteniidae, Megalopodidae, Oxypeltidae, Sinopyrophoridae
  plus 6 further subfamily-level conflicts — queried via both taxid AND parent-family taxid

## Locus naming convention

SuperCRUNCH uses exact locus name strings from FASTA headers. The names used here:
`COI`, `16S`, `18S`, `28S`, `CAD`, `EF1a`, `ArgK`, `RNApol2`, `wingless`
These must match the strings in `locus_names.txt` and reference FASTA headers.
Confirm against SuperCRUNCH's Parse_Loci.py output before full run.
