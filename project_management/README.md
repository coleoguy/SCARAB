# project_management/

Tracking files for the SCARAB project. All AI contributions, progress, and project decisions are documented here.

## Canonical Files

| File | Purpose | Format | Status |
|------|---------|--------|--------|
| `ai_use_log.csv` | **Primary** AI contribution log | CSV (machine-readable) | Current (31 entries) |
| `ai_use_log.md` | Detailed AI contribution log | Markdown (human-readable) | Current (29 entries — needs entries 30-31) |
| `ai_use_log.xlsx` | Excel snapshot of AI log | Excel | Stale — use .csv instead |
| `progress_tracking.md` | Detailed progress by session | Markdown | Current |
| `task_board.html` | Interactive drag-and-drop task board | HTML | Current |
| `timeline.md` | Phase-by-phase timeline | Markdown | Current |
| `postit_guide.md` | Quick notes and blockers | Markdown | Reference only |

## Other Tracking Files

## Notes

The **canonical AI use log** is `ai_use_log.csv` — it is always the most up-to-date. The `.md` version provides a more detailed narrative but may lag by 1-2 entries. The `.xlsx` version is a snapshot and should not be relied upon for current status.
