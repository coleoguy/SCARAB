3. ADEPHAGA MONOPHYLY: VERIFIED
   Status: MONOPHYLETIC (Perfect recovery)
   - Tips: 48/48 in group form a clade with no intruders
   - All Adephaga families recover together
   - Includes: Carabidae, Dytiscidae, Gyrinidae, and outgroups

   KEY IMPLICATION: Adephaga boundaries are well-resolved and correct.

4. POLYPHAGA STRUCTURE: SEVERELY PROBLEMATIC
   Status: NOT MONOPHYLETIC
   - Tips in group: 412
   - Tips in MRCA clade: 461
   - Intruders: 49
   - All intruders are Adephaga (48 Carabidae + 1 Dytiscidae)

   INTERPRETATION:
   The tree does NOT recover Adephaga and Polyphaga as sister groups.
   Instead, core Adephaga (Carabidae) are phylogenetically nested between
   different Polyphaga groups. This is a significant STRUCTURAL ERROR in the
   tree topology.

   This is NOT a problem with the catalog—it's a phylogenetic signal problem
   in the alignment or tree inference.

================================================================================
INTERPRETATION & IMPLICATIONS
================================================================================

1. PHYLOGENETIC SIGNAL vs TAXON SAMPLING:

   The catalog mapping is perfect (100%), so this is NOT a data integration
   problem. Instead, the non-monophyly reflects REAL PHYLOGENETIC SIGNAL
   in the tree inference.

   Key observations:
   a) Adephaga IS monophyletic and well-recovered
   b) Polyphaga is NOT monophyletic (Adephaga nested inside)
   c) Within Polyphaga, some superfamilies are monophyletic (Scarabaeoidea,
      Staphylinoidea) while others are highly fragmented (Curculionoidea,
      Tenebrionoidea)

2. THE ADEPHAGA-POLYPHAGA PROBLEM:

   The most serious issue is that Adephaga and Polyphaga do not form the
   expected clade structure. If these are true sister groups (as modern
   taxonomy holds), the tree has a TOPOLOGY ERROR.

   Possible causes:
   - Insufficient phylogenetic signal in alignments for these deep nodes
   - Taxon sampling artifacts (some Adephaga lineages may be attracting
     certain Polyphaga clades)
   - Alignment quality issues
   - True evolutionary signal (less likely given modern systematics)

3. WITHIN-POLYPHAGA STRUCTURE:

   The checkerboard pattern (Curculionoidea = highly fragmented, while
   Scarabaeoidea = monophyletic) suggests:
   - Some superfamilies are well-separated by phylogenetic signal
   - Others (especially Curculionoidea and Tenebrionoidea) have weak
     inter-superfamily signal or taxon sampling artifacts

   This is typical of genomic analyses where gene tree heterogeneity
   can create spurious relationships, especially when taxa are fragmentary.

4. POSITIVE FINDINGS:

   Despite the problems above:
   - Within-family structure is largely correct (66% of families monophyletic)
   - Within-genus structure is largely correct (83% of genera monophyletic)
   - Adephaga is perfectly recovered
   - Coleoptera is nearly perfectly monophyletic (99.8%)
   - Several superfamilies are well-recovered (Scarabaeoidea, Staphylinoidea,
     Cleroidea, Buprestoidea)

================================================================================
RECOMMENDATIONS
================================================================================

1. INVESTIGATE THE ADEPHAGA-POLYPHAGA TOPOLOGY:
   - Rerun phylogenetic inference with constrained topology
   - Test if alternative topologies have significantly higher likelihood
   - Check for long-branch attraction artifacts

2. EXAMINE SUPERFAMILY BOUNDARIES:
   - Focus on Curculionoidea and Tenebrionoidea fragmentation
   - These may reflect real biological divergences or alignment problems
   - Consider re-aligning or filtering genes with weak signal

3. VALIDATE WITH ALTERNATIVE METHODS:
   - Phylogenetic reconstruction with different inference methods
   - Concordance analysis across individual genes
   - Test for incomplete lineage sorting signal

4. EXPANSION OF TAXON SAMPLING:
   - Current tree is only 478/1121 tips (42.6%)
   - Some families are severely undersampled (Attelabidae: 2 tips)
   - Full tree with all 1121 taxa may resolve some of these issues
